
# Steps 
##### 1. composer update
##### 2. php artisan key:generate
##### 3. php artisan migrate:fresh --seed
##### 4. php artisan db:seed
##### 5. php artisan storage:link
##### 6. php artisan optimize:clear
##### 6. php artisan serve
##### 7. php artisan queue:work
##### 8. php artisan schedule:work
